## Tips

- GIF format for animated mascots
- PNG format for static mascots
- Keep file sizes under 1MB for animations
- Recommended dimensions: 400x400px
- Use a consistent style across all mascots
- If no mascot images uploaded, emoji fallbacks will display

## Creating Mascot Images

Options for creating mascots:
- Commission an illustrator
- Use AI image generation tools (Midjourney, DALL-E)
- Hire from Fiverr or similar platforms
- Use Lottie animations for smooth web animations
