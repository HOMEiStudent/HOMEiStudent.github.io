## Tips

- Use square images (1:1 ratio)
- Professional but friendly photos work best
- PNG or JPG format
- Keep file size under 500KB each
- Consider using animated/illustrated versions for a fun, approachable look

## Adding New Team Members

To add a new team member:
1. Upload their photo with a simple filename (e.g., `newname.png`)
2. Edit `about.html` to add a new team card
3. Reference the image as `images/team/newname.png`
