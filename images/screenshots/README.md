## Recommended Specifications

- **Format:** PNG (preferred) or JPG
- **Dimensions:** 1170 x 2532px (iPhone 14 Pro size) or similar
- **Aspect Ratio:** Approximately 9:19.5 (standard iPhone)
- **File Size:** Under 1MB each for fast loading

## Tips

- Use actual app screenshots for authenticity
- Choose screens that showcase key features
- Ensure text is readable at smaller sizes
- The screenshots auto-rotate every 4 seconds on the homepage

## Adding More Screenshots

To add more than 3 screenshots:
1. Upload additional files as `screenshot-4.png`, `screenshot-5.png`, etc.
2. Edit `index.html` to add more `<img>` tags in the phone-screen section
3. Add corresponding dots in the screenshot-dots section
