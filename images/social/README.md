# Social Media Icons

Upload your social media icons here. Each should be a PNG with transparent background.

## Required Files

| Filename | Description | Recommended Size |
|----------|-------------|-----------------|
| `instagram.png` | Instagram logo icon | 64x64px |
| `tiktok.png` | TikTok logo icon | 64x64px |
| `twitter.png` | X (Twitter) logo icon | 64x64px |
| `linkedin.png` | LinkedIn logo icon | 64x64px |

## Notes
- Use white or light-coloured versions of the logos (they'll sit on the dark footer background)
- PNG format with transparent background works best
- If you don't have a particular social account yet, just don't upload that icon and it will be hidden automatically
